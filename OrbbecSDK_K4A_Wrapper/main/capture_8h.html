k4a_result_t capture_create ( k4a_capture_t *  capture_handle)

Create a reference counted k4a_capture_t for tracking synchronized images.

Parameters
capture_handleThe capture handle

/ref k4a_capture_t object this API creates is with a base ref count of 1. It will automatically be deleted when its ref count reaches zero. Any k4a_image_t's associated with this capture will also have a reference removed when this object is destroyed.
