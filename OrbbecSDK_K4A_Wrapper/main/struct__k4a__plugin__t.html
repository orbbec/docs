Detailed Description

Plugin API which must be populated on plugin registration.

Remarks
The Azure Kinect SDK will call k4a_register_plugin, and pass in a pointer to a k4a_plugin_t. The plugin must properly fill out all fields of the plugin for the Azure Kinect SDK to accept the plugin.
