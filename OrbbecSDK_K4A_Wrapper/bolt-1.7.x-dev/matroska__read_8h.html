◆ find_next()

template<typename T >
std::unique_ptr< T > k4arecord::find_next ( k4a_playback_context_t *  context,
bool  search = false 
)

Find the next element of type T at the current file offset. If search is true, this function will keep reading elements until an element of type T is found or EOF is reached. If search is false, this function will only return an element if it exists at the current file offset.

Example usage: find_next<KaxSegment>(context, true);
